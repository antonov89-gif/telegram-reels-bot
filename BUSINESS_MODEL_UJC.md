# Бизнес-модель UJC Creator: Instagram Reels → Wildberries → Прибыль

## Цель: Больше показов артикула = больше заказов на WB = больше %

### Воронка (было → стало):

**БЫЛО (как ты делал вручную):**
Ты → сам анализируешь → сам пишешь промпт → сам генеришь видео в Gemini Veo → сам постишь в Instagram → руками пишешь артикул в описание

Проблема: тратишь 2 часа в день, постишь нерегулярно, артикул теряется.

**СТАЛО (новая схема с ботом):**

```
[1] АГЕНТ-АНАЛИТИК (Gemini 3.5) 24/7
    ↓ анализирует @ksusha.prettycutie и конкурентов
    → дает 3 вирусных идеи с хуками ВЫКИДЫВАЕМ БЕСПОЛЕЗНЫЕ БАНОЧКИ

[2] АГЕНТ-ПРОМПТ (Gemini)
    ↓ генерит готовый промпт для Veo Pro
    → ты копируешь 1 кликом и вставляешь в свое приложение Gemini Pro Veo
    → получаешь ПРОФЕССИОНАЛЬНОЕ видео (твое качество, не мое)

[3] ТЫ — кидаешь готовое видео в Telegram бот @VideoKsu222_bot
    ↓ бот спрашивает: "Какой артикул WB?"
    → ты пишешь: "12345678 - Сыворотка Сияние"
    → бот сохраняет в очередь

[4] АГЕНТ-АВТОПОСТИНГ (Instagrapi + Scheduler)
    ↓ хранит очередь: video.mp4 + артикул + подпись
    → постит по расписанию: 10:00, 15:00, 19:00 МСК (пик активности)
    → в видео вжигает текст: "АРТИКУЛ 12345678 НА WB ⬇️"
    → в подпись: "АРТИКУЛ ЗАБИРАЙ В ОПИСАНИИ 12345678 #ujccreator"
    → в сторис автоматом делает репост с ссылкой-стикером на WB
```

### Где вставлять артикул (чтобы люди вводили на WB):

1. **В самом видео (самое важное):**
   - Нижняя треть: `АРТИКУЛ 12345678 НА WB` — белый жирный с черной обводкой (как у Ксюши)
   - Люди скринят видео, артикул остается на скрине

2. **В описании Reels:**
   - Первая строка: `ВЫКИДЫВАЕМ БЕСПОЛЕЗНЫЕ БАНОЧКИ 🗑️`
   - Вторая: `АРТИКУЛ 12345678 👇`
   - Потом польза + CTA
   - Хештеги: #ujccreator #wbнаходки #артикулвб

3. **В шапке профиля:**
   - `UJC Creator | Ищи на WB по артикулу из Reels 👇`

4. **В Highlights:**
   - Папка "АРТИКУЛЫ" — скрины карточек WB с артикулами

5. **В сторис:**
   - Автопостинг агент после Reels постит сторис с тем же видео + стикер "Ссылка" на прямую ссылку WB: `https://www.wildberries.ru/catalog/12345678/detail.aspx`

### Как увеличить прибыль:

- **Частота:** 3 Reels в день по расписанию = 90 в месяц. Алгоритм Instagram любит регулярность → больше охватов артикула.
- **Тест артикулов:** пости один и тот же продукт с разными хуками: "В 30 лет ни морщинки" vs "Выкидываем бесполезные баночки" — смотри какой дает больше переходов на WB (в ЛК WB видно источник трафика).
- **Связка с WB отзывами:** когда на WB появляется отзыв "увидела в Instagram" — делай Reels с этим отзывом.

### Что я сейчас доделал в боте:

- Агент аналитики и промптов остался — генерит идеи и промпты для твоего Veo Pro
- Агент автопостинга теперь принимает видео от тебя: кидаешь файл → бот спрашивает артикул → ставит в очередь
- Scheduler: 10:00, 15:00, 19:00 МСК (можешь поменять в .env)
- Бот 24/7 на GitHub Actions — очередь не пропадет

Ты делаешь красивое видео сам (качество Veo Pro), а рутину (выжигание артикула, подпись, постинг по времени, сторис) делает бот.
